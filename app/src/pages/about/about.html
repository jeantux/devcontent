Devcontent

Esse site reúne os melhores conteúdos gratuitos sobre tecnologia na internet.
Todo os canais para participar da lista tem que possuir conteúdos disponíveis em alguma plataforma de comunicação e obrigatoriamente ter uma conta no github, isso porque os algorítimos do nosso site utilizam a API do github para buscar diversas informações sobre o produtor de conteúdo.

O projeto iniciou em uma live na twitch.tv onde o Jean Carlos desenvolveu uma versão minificada do devcontent chamada, devcontentBR, e depois foram realizadas diversas alterações para colocar em produção.

Esse conteúdo é gratuito e fica livre para contribuição, o único objetivo dessa plataforma é espalhar o conhecimento juntando os melhores canais do youtube.


A seleção do conteúdo é de forma manual, isso porque realmente precisamos validar as informações e conteúdos de cada canal.
A plataforma vai ter diversas melhorias, e um dos principais focos com certeza vai ser nos algoritimos de seleção e recomendação de canais.
